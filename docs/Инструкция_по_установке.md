# Инструкция по установке

**Игры для МБУК «Гайский историко-краеведческий музей»**
- Игра-стройка «Комсомольская ударная. Строим Гай» — `gay-stroika-prototype.html`
- Викторина «Большое путешествие по маленькому городу» — `viktorina.html`

Обе игры **полностью офлайн**: интернет не нужен ни для установки, ни для работы.
Работают в любом современном браузере (**Edge / Chrome / Firefox**).

> **Ноутбук музея:** MSI Thin 15 B12UC — **Windows 11 Pro**, Intel i5-12450H, 16 ГБ, RTX 3050.
> Ниже — раздел **A) для Windows** (основной, под этот ноутбук) и **Б) для Astra Linux** (если понадобится).

---

## A) Установка на Windows 10/11 (ноутбук музея)

**Быстрый запуск (без киоска):** откройте `gay-stroika-prototype.html` или `viktorina.html`
двойным кликом — игра откроется в браузере. **F11** — полный экран.

**Полноэкранный киоск:**
1. Скопируйте всю папку игры на ноутбук (например `C:\Гай\`).
2. Запустите `kiosk\start-kiosk.bat` — игра-стройка откроется в киоске
   (Edge/Chrome/Firefox — что установлено). Для викторины:
   `kiosk\start-kiosk.bat viktorina.html`.
   - Выход — `Alt`+`F4` (окно само перезапустится); чтобы остановить совсем —
     закрыть чёрное окно `.bat`.

**Автозапуск при включении:** двойной клик по `kiosk\install-autostart-windows.bat` —
создаст ярлык в папке автозагрузки. После перезагрузки игра стартует сама.
Для автозапуска викторины: в командной строке `install-autostart-windows.bat viktorina.html`,
или в свойствах ярлыка «Гай-киоск» (папка `shell:startup`) допишите `viktorina.html` в поле «Объект».
Отключить автозапуск — удалить ярлык «Гай-киоск» из папки `shell:startup`.

## Б) Установка на Astra Linux (если игра пойдёт на Linux-машину)

Работает в браузере **Firefox** или **Chromium**.

---

## 1. Что в комплекте (на флешке)

```
gay-stroika-prototype.html   ← игра-стройка (можно открыть двойным кликом)
viktorina.html               ← викторина (можно открыть двойным кликом)
libs/three.min.js            ← 3D-движок (локально)
foto/                        ← фото к вопросам викторины (кладёт музей)
audio/                       ← (необязательно) музыка music.mp3, если захотите свою
kiosk/                       ← скрипты автозапуска и киоска
docs/                        ← эта инструкция и руководства пользователя
```

> Вся 3D-графика игры-стройки — **процедурная** (рисуется кодом), сторонних
> 3D-моделей нет, поэтому игра лёгкая и грузится за секунду.

## 2. Копирование на ноутбук

Скопируйте **всю папку игр целиком** с флешки на ноутбук, например в домашнюю
папку: `/home/<пользователь>/gay-games/`.
**Не разбивайте папку** — `libs/` и остальные подпапки должны лежать рядом с
HTML-файлами.

## 3. Быстрая проверка (без киоска)

Самый простой способ — открыть HTML-файл двойным кликом: игра запустится в окне
браузера. Для полного экрана нажмите **F11**.

Через терминал (рекомендуется для проверки автозапуска):
```bash
cd ~/gay-games
chmod +x kiosk/start-kiosk.sh kiosk/install-autostart.sh
./kiosk/start-kiosk.sh                 # игра-стройка в полном экране (киоск)
./kiosk/start-kiosk.sh viktorina.html  # викторина в полном экране
```
Проверить полную автономность можно, **отключив Wi-Fi/кабель** — игры работают
точно так же.

## 4. Автозапуск при включении (киоск)

Чтобы выбранная игра стартовала сама после входа в систему:
```bash
./kiosk/install-autostart.sh           # по умолчанию — игра-стройка
```
После перезагрузки игра откроется в полноэкранном киоске автоматически и будет
сама перезапускаться, если её закрыть (watchdog).

**Сменить игру в автозапуске** (например, на викторину): откройте
`~/.config/autostart/gay-kiosk.desktop` и в строке `Exec=` допишите имя файла:
`.../kiosk/start-kiosk.sh viktorina.html`.

**Отключить автозапуск:**
```bash
rm ~/.config/autostart/gay-kiosk.desktop
```

## 5. Подключение геймпадов (Xbox/XInput по Bluetooth)

1. Включите **Bluetooth** в Astra Linux.
2. На геймпаде зажмите кнопку **Connect/Pair** до мигания.
3. Подключите его в настройках Bluetooth Astra.
4. Геймпады определяются играми **автоматически**, по порядку подключения:
   1-й подключённый = Игрок/Команда 1, 2-й = Игрок/Команда 2 и т.д.
   - Игра-стройка — до **4 геймпадов**; викторина — **2** (две команды).
   - Реально одновременно работают 4–6 геймпадов (предел Bluetooth-радио).
5. Если геймпада нет — всё управляется с **клавиатуры** (см. руководства пользователя).

## 6. Управление киоском (для персонала)

- **Выход из игры:** `Alt`+`F4`. По умолчанию `start-kiosk.sh` перезапустит игру
  через 2 секунды. Чтобы остановить совсем, в терминале:
  ```bash
  pkill -f start-kiosk ; pkill -f "\.html"
  ```
- **Звук:** клавиша **`M`** — включить/выключить звук в обеих играх.
- Правый клик, контекстное меню и клавиши `F5`/`F11`/`Ctrl+R`/`Ctrl+W`
  заблокированы внутри игр, чтобы дети случайно не вышли. Курсор прячется через 3 с.

## 7. Если что-то не так

- **Пустое/чёрное окно:** проверьте, что папка `libs/` лежит рядом с HTML-файлом.
- **«Не найден браузер»:** установите Firefox или Chromium (в офлайне — из
  репозитория Astra на установочном носителе).
- **Экран гаснет/уходит в сон:** скрипт сам отключает заставку (через `xset`);
  если не помогло — отключите спящий режим в настройках электропитания Astra.
- **Геймпад не реагирует:** переподключите по Bluetooth; проверьте, что система
  его видит (`ls /dev/input/js*`).
- **Нет звука:** нажмите `M` (вдруг выключен); проверьте громкость системы.
  Звук стартует после первого действия игрока (так требуют браузеры).
