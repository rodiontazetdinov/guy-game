ФОТО ВИКТОРИНЫ И ФОН
====================
q1..q41 — фотографии к вопросам викторины (получены от музея 2026-07-02),
          пронумерованы по номеру вопроса. Используются автоматически.
fon.jpg — фоновое фото (палаточный городок первостроителей) для экранов.

Чтобы заменить фото вопроса N — положите новый файл foto/qN.<jpg|png>
(и при смене расширения поправьте путь photo в блоке QUIZ_SETS в viktorina.html).
