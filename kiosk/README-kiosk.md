# Установка игр для музея г. Гай в режиме киоск (Astra Linux)

Игры полностью **офлайн** — интернет не нужен ни для установки, ни для работы.
Запускаются в браузере **Firefox** или **Chromium** в полноэкранном режиме.

Два самодостаточных файла-игры (открываются и двойным кликом):
- **`gay-stroika-prototype.html`** — игра-стройка «Комсомольская ударная. Строим Гай».
- **`viktorina.html`** — викторина «Большое путешествие по маленькому городу» (2 команды = 2 геймпада).

## 1. Что в комплекте

```
gay-stroika-prototype.html   ← игра-стройка (двойной клик)
viktorina.html               ← викторина (двойной клик)
panel-house-data.js          ← 3D-модель дома для стройки (встроена)
libs/                        ← three.js + загрузчик моделей (для стройки)
foto/                        ← фото к вопросам викторины (кладёт музей; без них — заглушка)
kiosk/
  start-kiosk.sh             ← запуск в полноэкранном киоске
  install-autostart.sh       ← включить автозапуск при входе в систему
  gay-kiosk.desktop          ← шаблон автозапуска (правит install-autostart.sh)
  README-kiosk.md            ← этот файл
```

## 2. Копирование с флешки

Скопируйте **всю папку игры целиком** на ноутбук, например в домашнюю папку:
`/home/<пользователь>/gay-stroika/`. Не разбивайте папку — `libs/` и
`panel-house-data.js` должны лежать рядом с `gay-stroika-prototype.html`.

## 3. Быстрая проверка (без автозапуска)

1. Откройте терминал в папке игры.
2. Сделайте скрипт исполняемым и запустите нужную игру:
   ```bash
   chmod +x kiosk/start-kiosk.sh
   ./kiosk/start-kiosk.sh                 # игра-стройка (по умолчанию)
   ./kiosk/start-kiosk.sh viktorina.html  # викторина
   ```
   Откроется полноэкранное окно с заставкой игры.
3. Стройка — **НАЧАТЬ СТРОЙКУ**; викторина — выберите набор вопросов
   (мышью или кнопкой **A** / <span>Пробел</span> на геймпаде/клавиатуре).

Проверить совсем без сети можно, отключив Wi-Fi/кабель — игра работает так же.

## 4. Автозапуск при включении

Чтобы игра стартовала сама после входа в систему:
```bash
./kiosk/install-autostart.sh
```
Скрипт пропишет автозапуск в `~/.config/autostart/`. После перезагрузки
ноутбука игра откроется в киоске автоматически.

**Отключить автозапуск:**
```bash
rm ~/.config/autostart/gay-kiosk.desktop
```

## 5. Геймпады (Xbox/XInput по Bluetooth)

- Включите Bluetooth в Astra Linux, переведите геймпад в режим сопряжения
  (кнопка Connect/Pair), подключите через настройки Bluetooth.
- Геймпады определяются игрой автоматически. **Игрок 1** — первый
  подключённый, **Игрок 2** — второй.
- Реально одновременно работают 4–6 геймпадов (предел Bluetooth-радио),
  оптимум — 4. Сейчас в игре-стройке заняты 2 (расширение до 4 — отдельная задача).
- Резерв — клавиатура:
  - **Стройка:** Игрок 1 — `WASD` + `Q/E` + `Пробел`/`X`; Игрок 2 — стрелки + `,`/`.` + `Enter`/`/`.
  - **Викторина:** Команда 1 — `W`/`S` выбор, `D` ответить; Команда 2 — `↑`/`↓` выбор, `Enter` ответить;
    ведущий — `Пробел` (показать ответ / следующий вопрос).

## 6. Управление киоском для персонала

- **Выход из игры:** `Alt`+`F4` закрывает окно. Скрипт `start-kiosk.sh`
  по умолчанию **перезапустит** игру через 2 секунды (watchdog) — чтобы
  полностью остановить, в терминале:
  ```bash
  pkill -f gay-stroika-prototype     # закрыть браузер игры
  pkill -f start-kiosk               # остановить watchdog-перезапуск
  ```
- Правый клик, контекстное меню и клавиши `F5`/`F11`/`Ctrl+R`/`Ctrl+W`
  заблокированы в самой игре, чтобы дети случайно не вышли.
- Курсор мыши прячется через 3 секунды бездействия.

## 7. Если что-то не так

- **Пустое/чёрное окно, нет 3D:** проверьте, что папка `libs/` и файл
  `panel-house-data.js` лежат рядом с `gay-stroika-prototype.html`.
- **«Не найден браузер»:** установите Firefox или Chromium
  (в офлайне — из репозитория Astra на установочном носителе).
- **Экран гаснет/уходит в сон:** скрипт сам отключает заставку и сон экрана
  (через `xset`). Если не помогло — отключите спящий режим в настройках
  электропитания Astra.
- **Геймпад не реагирует:** переподключите по Bluetooth; проверьте, что он
  определяется системой (`ls /dev/input/js*` или утилита проверки джойстика).
